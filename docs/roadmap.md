# Roadmap

## Этап 0 — Документация и направление

Цель: зафиксировать концепцию, базовые системы и архитектуру.

Готово в рамках текущей документации:

- описание концепции;
- описание строительной системы;
- описание энергии, жителей и баланса;
- описание архитектуры без монолита;
- описание визуального направления;
- предварительный план разработки.

## Этап 1 — Прототип строительства

Цель: проверить, приятно ли строить башню из домиков.

Минимальные задачи:

1. Создать основание.
2. Создать 5–10 тестовых домиков.
3. Реализовать скрытую логическую сетку или систему слотов.
4. Реализовать размещение домиков сверху.
5. Реализовать подсветку доступных мест.
6. Реализовать рост башни вверх.
7. Проверить, что визуально получается красиво.

На этом этапе не нужны сложные жители, энергия и прогрессия. Главное — ощущение сборки.

## Этап 2 — Выбор из трёх вариантов

Цель: добавить простое геймплейное решение.

Задачи:

1. После каждого строительства выдавать три новых варианта.
2. Сделать простые категории: жильё, кафе, библиотека, мастерская, ветряк.
3. Добавить базовые параметры зданий.
4. Сделать простые правила появления вариантов.
5. Проверить, не перегружает ли выбор игрока.

## Этап 3 — Энергия

Цель: добавить мягкое ограничение роста.

Задачи:

1. Добавить ветряки.
2. Добавить радиус энергии.
3. Отображать, какие дома обеспечены энергией.
4. Сделать так, чтобы дома без энергии хуже заселялись.
5. Добавить визуальную обратную связь через свет в окнах.

## Этап 4 — Жители и комфорт

Цель: оживить башню.

Задачи:

1. Добавить вместимость жилья.
2. Добавить появление жителей.
3. Добавить комфорт.
4. Связать комфорт с кафе, библиотеками, арт-студиями и магазинами.
5. Добавить маленькие анимации жителей.

## Этап 5 — Красота и технология

Цель: сделать прогрессию визуально уникальной.

Задачи:

1. Добавить параметры красоты и технологии.
2. Связать типы зданий с этими параметрами.
3. Настроить открытие новых форм зданий.
4. Сделать разные визуальные направления башни.
5. Проверить, что разные игроки получают разные башни.

## Этап 6 — Мостики, боковые элементы и острова

Цель: расширить выразительность башни.

Задачи:

1. Добавить боковые элементы: балконы, лестницы, ветряки.
2. Добавить мостики между двумя опорами.
3. Добавить старт с двух или трёх оснований.
4. Добавить более сложные формы домиков.

## Этап 7 — Share и replay

Цель: усилить вирусный потенциал.

Задачи:

1. Хранить историю строительства.
2. Генерировать короткий timelapse.
3. Добавить красивую камеру, которая показывает рост башни.
4. Добавить финальный экран с башней.
5. Подготовить экспорт видео или share-картинки.

## Этап 8 — Контент и полировка

Цель: превратить прототип в полноценную игру.

Задачи:

1. Добавить больше домиков.
2. Добавить стилистические наборы.
3. Настроить баланс прогрессии.
4. Улучшить анимации.
5. Добавить звук и музыку.
6. Добавить сохранения.
7. Подготовить мобильный UI.
